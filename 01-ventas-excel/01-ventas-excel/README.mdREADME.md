# 📈 Proyecto 1: Análisis de Ventas en Excel

## Objetivo
Identificar tendencias de ventas, productos más rentables y regiones con mayor crecimiento.

## Herramientas
`Excel` `Power Query` `Tablas dinámicas` `Gráficos`

## Análisis realizado
- Ventas totales por región
- Top 10 productos más vendidos
- Tendencias mensuales de ingresos
- KPIs: Ticket promedio, % crecimiento, margen

## Hallazgo principal
La región Norte concentra el 40% de las ventas. El Q4 es el trimestre de mayor ingreso.
